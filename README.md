# React & GraphQL

Simple example for fullstack React & GraphQL & MySQL.

Based on https://blog.logrocket.com/crud-react-graphql-examples/

### Installation

#### Server

Import SQL database to a MySQL DB with name `userapp`:

```
mysql> source server/userapp.sql
```

Run server through:

```
yarn install
yarn start
```

#### Client

Run through:

```
yarn install
yarn start
```
